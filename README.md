# ApplyPilot Career

ApplyPilot is a Firebase-hosted job application tracker. Users sign in, answer job-goal questions, save resume/profile details, track applications, score job descriptions, and prepare outreach messages.

## Live app

- https://applypilot-career.web.app

## Stack

- Firebase Hosting
- Firebase Authentication
- Cloud Firestore
- Static HTML, CSS, and JavaScript

## Deploy

```bash
firebase deploy --only hosting --project akul-applypilot-20260710
firebase deploy --only firestore:rules --project akul-applypilot-20260710
```
